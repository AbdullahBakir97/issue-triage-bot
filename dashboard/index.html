Features

🏷

Auto-Categorization

Detects issue type: bug, feature request, question, documentation, enhancement, or support.

🚨

Priority Detection

Assigns P0-P4 priority based on severity keywords like security, crash, or cosmetic.

✅

Completeness Check

Verifies bug reports include reproduction steps, expected behavior, and environment info.

🔍

Duplicate Detection

Finds potential duplicates using keyword overlap analysis against recent issues.

💬

Auto-Response

Posts template comments asking for missing info, thanking good reports, or noting duplicates.

⚙

Configurable

Per-repository configuration via .github/triage-bot.yml with custom labels and assignees.
